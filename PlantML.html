Summary:
This project is a machine trained model that identifies plant leaves in addition to determining if they are diseased or healthy. I used a Google Colab notebook with python, pandas, NumPy, and Matplotlib. The GUI was put in a web-based HTML and used pycharm and streamlit.

The robust dataset was 54,305 images divided into 38 classes. The data was preprocessed. Any anomalies, such as files without pictures or repeats, were removed. The images were resized to be uniform, and then split into an 80/20% training and validation set.

This project built a supervised classification model for images using Convolutional Neural Networks (CNNs), a fundamental method in contemporary image classification. CNNs are renowned for their ability to automatically extract and learn spatial features from images, making them the cornerstone of modern image classification techniques.

The architecture of the model was designed with sequential layers, forming a deep neural network. These layers form a deep neural network. The layers were strategically added in sequence, incorporating key components such as Conv2D layers with varying filter sizes, MaxPooling layers for down-sampling, Flatten layers to transform the data into a one-dimensional array, and Dense layers for classification.

The Conv2D layers convolve the input images with a set of learnable filters to capture relevant features. Subsequently, MaxPooling layers down-size the feature maps to reduce dimensionality and computation. The Flatten layer then transforms it into a one-dimensional array, enabling a seamless integration with Dense layers for classification purposes.

The model was developed iteratively, with data passing through the algorithm multiple times to refine the network's parameters and optimize its performance. The selection of CNNs for this project was justified by its proven effectiveness in handling images to automatically learn and extract meaningful features for image classification. Additionally, the sequential architecture eased the construction of a robust and scalable model suitable for classifying plant leaves.
