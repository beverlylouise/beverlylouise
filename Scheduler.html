Scheduling Application

Java HTML MySQL JavaFX IntelliJ IDEA SceneBuilder
Github Repository

Summary: This is a robust customer record and appointment scheduling application. The log-in form is displayed in English or French, based on the user’s computer language setting. All texts, labels, buttons, errors, and messages are translated. This uses the company’s MySQL database.

Tech Specs: The application was built using Java and JavaFX. The GUI was developed with SceneBuilder. There is a logger function that writes all log-in attempts to a text file. The data is pulled from the company’s MySQL database. The program contains lambda functions, streams, and filters to efficiently reduce the lines of code and manipulate data. Resource bundles are used for language translation. Time zones are accounted for as all times are stored in UTC but displayed in the user’s local time zone. Appointment times are checked against the business hours. The application incorporates advanced exception control mechanisms to improve user experience and application stability. The application is generously commented on using Javadoc so the code can be easily read and understood by other programmers.
